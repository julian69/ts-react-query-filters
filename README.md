### Description
This is a very simple project I built to find out what react-query was about.

### Stack
- React
- TypeScript
- React-query
- Tailwing
- Vite
- Vitest
- React testing library 
- Cypress

### Scripts
Run development server

```bash
$ npm run dev
```

Build

```bash
$ npm run build
```

Unit and Integration Test

```bash
$ npm run test
```

e2e Test

```bash
$ npm run cy:withServer
```
